
# تطبيق نظام إدارة النبطشيات - Nabatchy Pro

تم تحويل الموقع الإلكتروني إلى تطبيق موبايل متكامل باستخدام تقنية **Flutter**.

## المميزات:
- واجهة مستخدم تدعم اللغة العربية (RTL).
- إمكانية اختيار الشهر والسنة وتوليد الجدول تلقائياً.
- إضافة وحذف الموظفين من القائمة.
- حفظ البيانات محلياً على الهاتف باستخدام `shared_preferences`.
- حساب الإحصائيات (الشيفتات والساعات) لكل موظف.

## متطلبات التشغيل:
1. تثبيت [Flutter SDK](https://docs.flutter.dev/get-started/install).
2. محرر أكواد (VS Code أو Android Studio).

## طريقة التشغيل:
1. قم بفتح المجلد في محرر الأكواد.
2. قم بتشغيل الأمر التالي في التيرمينال لتحميل المكتبات:
   ```bash
   flutter pub get
   ```
3. قم بتوصيل هاتفك أو تشغيل المحاكي (Emulator).
4. قم بتشغيل التطبيق:
   ```bash
   flutter run
   ```

## هيكل المشروع:
- `lib/models.dart`: يحتوي على نماذج البيانات (Worker, ShiftAssignment).
- `lib/logic.dart`: يحتوي على منطق توليد الأيام وحساب الإحصائيات.
- `lib/main.dart`: يحتوي على واجهات المستخدم (UI) والتعامل مع التخزين المحلي.
